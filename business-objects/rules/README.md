# Business Rules (objekt-übergreifend)

Business Rules, die über ein einzelnes Business Object hinausgehen. Pro Regel eine Datei mit Schema `BR-NN-name.md`.

Objekt-spezifische Regeln werden direkt im jeweiligen Business Object als Sektion geführt.

## Format

- Rule-ID (BR-NN)
- Aussage in natürlicher Sprache
- Formale Aussage (falls möglich, z.B. JSON Logic, OCL)
- Auslöser (Event, Zustand)
- Betroffene Business Objects
- Compliance-Bezug (z.B. DSGVO Art. 6, FINMA-Rundschreiben)
- Beispiele
